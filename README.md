# tcoffee-nf

A DSL2 nextflow pipeline to run T-Coffee popular modes 

## T-Coffee server modes

The first modes to be implemented on this workflow will be T-Coffee server modes.

### Alignment
 
* T-Coffee
* M-Coffee
* R-Coffee
* SARA-Coffee
* Expresso
* PSI-Coffee
 
### Evaluation

* TCS
* iRMSD-APDB
* T-RMSD
* Strike
